# ETH WALLET

The environment will be set in file `run.sh`

```bash
INFURA_PROJECT_ID: This is your INFURA_PROJECT_ID.
INFURA_PROJECT_SECRET: This is your INFURA_PROJECT_SECRET
```

after configired run this by :

```bash
./run.sh
```
